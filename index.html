About Website

Joey DeGrand's Personal Website

This old-fashioned Windows themed website is a website about me, Joey DeGrand, created for fun to learn more about web development in general and to display my experiences/interests. It utilizes a style library 98.css to make the pop-ups look as they did on Windows 98.

Here are some features of this site:

  • Click the 'OK' button on the bottom of a window to dismiss it
  • Click a file name on the left to bring that window to the front of the page or reopen the window if it is closed
  • Click a folder name on the left to make all of its contents visible
  • Double click a file name to position it on the top left of the desktop
